================================================================================
  3-STEP SETUP (5 MINUTES)
================================================================================

STEP 1: Download
----------------
Link: https://nodejs.org/dist/v20.11.0/node-v20.11.0-win-x64.zip
Size: ~30 MB
Save to: Downloads folder

STEP 2: Extract
---------------
1. Find the downloaded ZIP file
2. Right-click â†’ "Extract All..."
3. Extract to: C:\Users\YourName\nodejs
   (Replace YourName with your Windows username)
4. Click "Extract"

STEP 3: Run Setup
-----------------
Open PowerShell in project folder:
```powershell
cd terminology-extraction
.\setup-windows.ps1
```

**Done!** Setup will auto-detect portable Node.js and continue.

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: Setup still says "Node.js not found"
Solution: Check extraction location

```powershell
# Check if node.exe exists
Test-Path "$env:USERPROFILE\nodejs\node.exe"
```

Should show: True

If False, re-extract to correct location.

Problem: ZIP won't extract
Solution: Use 7-Zip or WinRAR if Windows built-in extractor fails

Problem: "Access denied" when extracting
Solution: Extract to your user folder (C:\Users\YourName\) - no admin needed
