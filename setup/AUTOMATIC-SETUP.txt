================================================================================
  AUTOMATIC PORTABLE NODE.JS INSTALLATION
================================================================================

The setup script can now automatically download and install portable Node.js!

================================================================================
  HOW IT WORKS
================================================================================

When you run setup-windows.ps1:

1. Setup checks if Node.js is installed
2. If NOT found, asks: "Download portable Node.js? (Y/n)"
3. Press "Y" or just press Enter
4. Setup automatically:
   - Downloads Node.js v20.11.0 (~30 MB)
   - Extracts to C:\Users\YourName\nodejs
   - Adds to PATH
   - Verifies installation
   - Continues with setup

**No manual download needed!**
**No manual extraction needed!**
**No manual PATH configuration needed!**

================================================================================
  IF DOWNLOAD FAILS
================================================================================

**Common Issues:**

1. **Behind corporate proxy**
   - Setup may fail to download
   - Manual download option available
   - See: setup/INSTALL-NODEJS.txt

2. **Firewall blocks nodejs.org**
   - Download manually at home/phone
   - Transfer ZIP to company computer
   - Extract to C:\Users\YourName\nodejs
   - Run setup again

3. **Network timeout**
   - Run setup script again
   - Or download manually

**Manual Fallback:**
If automatic download fails, setup shows manual instructions:
1. Download ZIP from nodejs.org
2. Extract to user folder
3. Run setup again

================================================================================
  OFFLINE INSTALLATION
================================================================================

**For computers without internet:**

1. **On computer with internet:**
   - Download: https://nodejs.org/dist/v20.11.0/node-v20.11.0-win-x64.zip
   - Copy to USB drive

2. **On offline computer:**
   - Copy ZIP to Downloads folder
   - Extract to: C:\Users\YourName\nodejs
   - Run setup script
   - Setup auto-detects portable Node.js

================================================================================
  TROUBLESHOOTING
================================================================================

**Problem: "Download failed"**
Solution: Check internet connection, try again, or download manually

**Problem: "Extraction failed"**
Solution: Check disk space (need 100 MB free), try again

**Problem: "Access denied"**
Solution: Setup uses user folder (always writable), should not happen

**Problem: Download very slow**
Solution: Wait for completion, or press Ctrl+C and download manually

**Problem: "Cannot verify installation"**
Solution: Run setup again, or check C:\Users\YourName\nodejs\node.exe exists
